Capacitación de Especialidades

Especialidad-01

En esta especialidad tenemos capacitados de seguridad, que son aptos para la portacion de armas no letales (Porra, Bastón extensible, Aerosoles de defensa personal). Estos guardias son ideales para la protección de un barrio privado, seguridad de eventos, seguridad de hospitales/escuelas, entre otros.

Especialidad-02

En esta especialidad tenemos capacitados de seguridad, que son aptos para la portacion de armas no letales (Pistolas taser, Lanzadores de balas de goma, Escudos antidisturbios). Estos guardias son ideales para prestar un servicio de alta seguridad para un evento, guardias personales, seguridad de banco o de importancia.

Especialidad-03

En esta especialidad tenemos capacitados de seguridad, que son aptos para la portacion de armas letales (armas de fuego). Estos guardias son ideales para hacer transporte de valores, agente de seguridad para escoltar personas importantes, entre otros.
